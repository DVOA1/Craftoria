# エンチャント適性

バニラでは、アイテムのエンチャント適性は<Color id="dark_purple">Final Power</Color>に影響する。

Apotheosisでは、代わりに直接<Color id="dark_purple">アルカナ</Color>によって増加するよう仕組みが変化する。

<Color id="dark_purple">アルカナ</Color>はアイテムのエンチャント適性を50%増加させるようになる。
